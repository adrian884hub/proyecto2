# Proyecto QA Testing - Web App

## 📌 Descripción
Este proyecto consiste en el testing manual de una aplicación web, donde se validaron funcionalidades principales y se identificaron posibles errores.

## 🧪 Actividades realizadas
- Diseño de casos de prueba
- Ejecución de testing manual
- Identificación y reporte de bugs
- Validación de interfaz y comportamiento

## 🛠 Herramientas utilizadas
- Navegador web
- GitHub
- Testing manual

## 📂 Archivos incluidos
- test-cases.md → Casos de prueba
- bug-reports.md → Reporte de errores

## 📈 Objetivo
Simular tareas reales de un QA Tester Junior aplicando buenas prácticas de testing.
